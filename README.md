# TestSystem
一款基于SSM+Vue+Element的在线考试平台

## 用户角色：

> 教师：
>
> - 管理班级成员，实现简易增删改查
>
> - 管理学科，实现简易增删改查
>
> - 创建题库，并在题库中选题实现创建试卷
>
> - 管理试卷
>
> 学生：
>
> - 在试卷中心查看所有试题，可以点击对应试题进入考试
> - 可以在用户中心查看个人信息，退出登录
> - 可以查看自己的考试记录
